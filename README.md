# 🎯 Jogo da Forca

Este é um jogo da forca simples e funcional, desenvolvido com **HTML, CSS e JavaScript**.  
O projeto foi criado para treinar lógica de programação,

---
## 🚀 Funcionalidades

- Exibição da palavra oculta
- Sistema de tentativas (vidas)
- Boneco da forca aparecendo a cada erro
- Dicas para ajudar o jogador
- Sistema de níveis (Fácil, Médio e Difícil)
- Pontuação por acertos
- Totalmente responsivo (funciona no celular, tablet e notebook)

---
## 🛠 Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript

---
## 📱 Responsividade

O jogo se adapta automaticamente a diferentes tamanhos de tela, garantindo boa experiência em dispositivos móveis e desktop.

---
## ▶️ Como executar o projeto

1. Faça o download ou clone o repositório
2. Abra a pasta do projeto
3. Clique duas vezes no arquivo `Index.html`
4. O jogo abrirá no navegador
